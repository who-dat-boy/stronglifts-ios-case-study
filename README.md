# stronglifts-ios-case-study

iOS case study for StrongLifts: a small SwiftUI interaction improvement for workout logging flow.

## Why I Built This
After using the app myself, it took me quite a lot of time to understand the workout screen flow (especially set and reps logging).  
So I built a simple fix based on my own interpretation of how this interaction could feel clearer and faster.

## What I Changed
- Added a more explicit set-completion interaction with visual continuity.
- Improved the transition from active set input to completed set state.
- Focused on clarity first, while keeping the interaction lightweight and fast.

## Tech
- Swift
- SwiftUI
- `matchedGeometryEffect`
- Gesture-driven input flow
